# Mobile-Sales-Performance-Dashboard

# Project Overview
This project is an interactive Power BI dashboard built to analyze mobile sales performance across different cities, brands, and time periods.
It provides insights into sales trends, customer behavior, and payment preferences.

# Problem Statement
Businesses often struggle to track sales performance across regions, products, and customer segments.

This project aims to simplify data into actionable insights for better decision-making.

# Tools Used
Power BI - Dashboard creation & visualization

Excel – Data cleaning and preprocessing


# Key Metrics
Total Sales: 308M

Total Quantity Sold: 8K

Total Transactions: 2K

Average Sales Value: 41K

# Key Insights
Major sales are concentrated in key metropolitan cities

Certain brands (like Apple & OnePlus) contribute higher revenue

Sales remain relatively stable across months with slight fluctuations

Credit cards and UPI are the most preferred payment methods

Higher ratings are more frequent, indicating overall positive customer feedback

# Recommendations
Focus marketing efforts on high-performing cities

Strengthen partnerships with top-selling brands

Promote digital payment options for smoother transactions

Analyze low-performing regions to improve sales

Maintain product quality to sustain high customer ratings
